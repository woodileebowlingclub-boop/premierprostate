# Premier League Charity Challenge

A browser-only fundraising ticket draw app for a Premier League charity challenge supporting prostate cancer research.

Open `index.html` to run the app locally, or host the repository with GitHub Pages, Netlify, or Vercel.

## Public Participant Page

Share `public.html` with participants so they can view progress without admin controls.

To update hosted public data:

1. Open the admin app.
2. Go to Admin.
3. Click **Export Public Data**.
4. Upload/replace `public-data.json` in the hosted repository.
